Book Review
=========

This is a forum app, based on Ruby on Rails.




Database
--------

This application uses SQLite with ActiveRecord.

Development
-----------

-   Template Engine: ERB
-   Testing Framework: Test::Unit
-   Front-end Framework: Bootstrap 3.0 (Sass)
-   Form Builder: SimpleForm
-   Authentication: Devise
-   Authorization: None
-   Admin: None
